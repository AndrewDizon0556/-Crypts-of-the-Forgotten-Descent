# -Crypts-of-the-Forgotten-Descent
A turn-based roguelike dungeon crawler built with Python and pygame. Features procedural dungeon generation (BSP), permadeath, enemy AI with FSM, 3 character classes, a multi-phase boss fight, inventory system, FOV mechanics, and a SQLite leaderboard. Descend 10 floors. Collect 7 shards. Survive.
